# Calculator

Pequeña calculadora hecha con **HTML**, **CSS** y **JavaScript** para practicar desarrollo front end. Permite realizar operaciones básicas (suma, resta, multiplicación y división).
Small calculator made with **HTML**, **CSS**, and **JavaScript** for practicing front-end development. It allows you to perform basic operations (addition, subtraction, multiplication, and division).

## Features

- Simple and responsive interface.
- Basic operations: addition, subtraction, multiplication, and division.
- Simple input validation (prevents invalid operations).
- Project designed as practice and part of my portfolio.

## How to use it

1. Open the `index.html` file in your browser.
2. Enter the numbers in the corresponding fields.
3. Select the operation.
4. Click the calculate button to see the result.

## Technologies used

- HTML5
- CSS3
- JavaScript (vanilla)
